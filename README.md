# E-Waste Management System

This project is a web application developed using Flask and SQLite to manage e-waste pickup requests.

## Features
- User Registration & Login
- Submit Pickup Requests
- Admin Dashboard
- Employee Assignment
- Request Tracking

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS

## How to Run
1. Install Python
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
